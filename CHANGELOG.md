# История изменений

## v0.4.0 - 2026-07-17

### Добавлено

- команда `/system` с информацией о CPU, RAM, диске, аптайме и процессе бота;
- команда `/log` для скачивания файловых логов ZIP-архивом;
- раздел `Настройки → Системные настройки`;
- включение и выключение глобального логирования;
- настраиваемый лимит размера логов и автоочистка раз в час;
- принудительная очистка логов из Telegram;
- Linux-скрипт `start.sh`;
- готовый unit `deploy/moneybot.service` для systemd;
- UTF-8 окружение для корректных русских журналов на Linux.

### Изменено

- кнопки `Ответ на сообщения` и `Ответ на отзывы` размещены в одной строке;
- основной журнал дополнительно сохраняется в `bot/data/moneybot.log`.

### Удалено

- тестовая команда `/testmessage`.

## v0.3.0 - 2026-07-17

### Добавлено

- раздел `Аккаунт → Лоты`;
- экспорт активных лотов FunPay в CSV-файл;
- настройка текста автовыдачи отдельно для каждого ID лота;
- переменные автовыдачи `{username}`, `{order_id}`, `{lot}`, `{sum}`, `{amount}`, `{account}`;
- автоматическая отправка настроенного текста покупателю при новом оплаченном заказе;
- сохранение обработанных заказов для защиты от повторной выдачи;
- Telegram-уведомление о результате автовыдачи.

## v0.2.0 - 2026-07-17

### Добавлено

- публичная документация по установке и настройке;
- безопасный `config.example.json` в корне проекта;
- совместимость с новой HTML-разметкой сообщений FunPay.

### Изменено

- кнопки `Тикеты` и `Настройки` поменяны местами в главном меню;
- уведомления о собственных исходящих сообщениях FunPay по умолчанию отключены;
- рабочий `config.json`, логи и runtime-данные исключены из Git.

### Исправлено

- доставка входящих сообщений FunPay в Telegram после изменения класса `chat-msg-text`;
- отправка ответов из Telegram в FunPay после обновления `PHPSESSID` и CSRF-токена;
- ложное сообщение `Ответ не отправлен`, когда FunPay уже принял ответ;
- разбор истории чатов на актуальной версии сайта FunPay.

## v0.1.0 - 2026-07-17

- первая рабочая версия MoneyBot;
- управление FunPay-аккаунтом из Telegram;
- уведомления, автоответы на сообщения и отзывы;
- тикеты FunPay Support;
- поддержка совместимых Cardinal-плагинов.
